# FiapBurgerIdentity

Uma API desenvolvida para abstrair o processo de autenticação de usuários no AWS Cognito.

## Stack
- Typescript
- Docker
- Terraform
- AWS Cognito
- AWS Lambda
- AWS API Gateway


Maiores detalhes sobre todo o processo ver no repositório da aplicação principal:
